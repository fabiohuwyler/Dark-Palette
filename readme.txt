Create a custom dark palette for your WordPress block theme and let visitors switch between Light, Dark and Auto modes.

== Description ==

Dark Palette brings native dark mode to modern WordPress block themes.

Instead of replacing stylesheets or inverting colours, it works with the colour presets generated by theme.json. Define a dark counterpart for each colour in your active theme palette, then add the Appearance Toggle block wherever visitors should be able to choose Light, Dark or Auto mode.

Features:

* Custom dark counterparts for the active theme colour palette
* Original and dark colours displayed side by side in the settings
* Light, Dark and Auto appearance modes
* Respects the visitor's system colour-scheme preference
* Remembers explicit choices in localStorage
* Applies the correct appearance before the page paints
* Accessible controls with keyboard and pressed-state support
* Optional labels and Auto mode in the Appearance Toggle block
* Per-post and per-page dark-mode exclusions in the editor
* Global exclusions for templates, archives, search, 404 pages, password-protected content, WooCommerce checkout and WordPress login
* Configurable unavailable-state message or disabled toggle
* No jQuery and no stylesheet swapping

Dark Palette was originally designed around Twenty Twenty-Five, but its palette-based approach is intended for modern block themes that use WordPress colour presets.

== Installation ==

1. Upload the dark-palette folder to /wp-content/plugins/, or install the ZIP through Plugins > Add New > Upload Plugin.
2. Activate Dark Palette.
3. Open Appearance > Dark Palette and define the dark counterpart for each colour.
4. Open Appearance > Editor and add the Appearance Toggle block to a header, footer or template part.
5. Save your changes.

== Frequently Asked Questions ==

= Does Dark Palette work with classic themes? =

Dark Palette is designed for block themes that expose their palette through theme.json and WordPress preset colour variables. Classic themes may not provide the variables the plugin expects.

= Can I disable dark mode for one page? =

Yes. Open the page or post editor and use the Dark Palette panel in the Document settings sidebar.

= What happens when dark mode is unavailable? =

You can hide the Appearance Toggle, show a custom message, or display a disabled toggle with an explanation. A global default can be overridden per post or page.

= Does it support WooCommerce? =

Dark Palette can keep the WooCommerce checkout in light mode when WooCommerce is active. It does not otherwise alter WooCommerce styles.
